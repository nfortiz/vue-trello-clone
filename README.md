# trello-clone
A trello clone build with vue, vuex, vue-router and vuetify. 
## Project setup
Install all project dependencies.
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
